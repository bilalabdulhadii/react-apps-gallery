## Overview

**Apps Gallery** is a dynamic web application designed to serve as a centralized portfolio for developers. It provides a polished, professional interface to display projects, complete with detailed descriptions, screenshots, and direct links to source code or live demos.

Built with a focus on **User Experience (UX)**, the app features a seamless **Dark/Light mode**, glassmorphism UI elements, and a responsive masonry layout that adapts to any device size.

## Installation & Setup

Follow these steps to run the project locally:

1.  **Clone the repository**

    ```bash
    git clone https://github.com/bilalabdulhadii/react-apps-gallery.git
    cd react-apps-gallery
    ```

2.  **Install dependencies**

    ```bash
    npm install
    ```

3.  **Start the development server**

    ```bash
    npm start
    ```

    The app will run at `http://localhost:3000`.

4.  **Build for production**
    ```bash
    npm run build
    ```

## Usage Guide

1.  **Browse:** Scroll through the masonry grid to view all available projects.
2.  **Search:** Use the search bar in the header to filter apps by **Title** or **Tags** (e.g., "React", "Tools").
3.  **View Details:** Click on any project card to view its full description, screenshots, and developer info.
4.  **Interact:** Click on screenshots to open the lightbox viewer; use arrow keys to navigate.
